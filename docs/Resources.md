# Resourcen

## Grundresourcen

- Geld
- Stein
- Stahl
- Treibstoff

## Energiebilanz

- Strom-Produktion - Strom-Verbauch ist Strombilanz.
- Strombilanz muss immer positiv sein, sonst kein weiterer Gebäudebau möglich
- Gebäude können nicht gebaut werden wenn nicht genug Strom vorhanden ist.

## Einwohner

- Unterkunfts Gebäude stellen Einwohner
- Eventuell späterer nutzen
- Einwohner zahlen später Steuern für extra Einkünfte
